Step-by-Step Guide to Upload and Host Your Website on GitHub Pages

  3. Name the repository something relevant (e.g., fusionfinancetech).
  4. Check the option “Add a README file”.
  5. Click Create repository.

3. Upload Your Website Files

  1. In your repository, click the “Add file” button and choose “Upload files”.
  2. Upload your index.html, about.html, services.html, contact.html, and styles.css files to the root of the repository.
  3. Once you’ve selected all the files, click "Commit changes" at the bottom of the page.

4. Enable GitHub Pages

  1. In the repository, click on the “Settings” tab.
  2. Scroll down to the “Pages” section in the left sidebar.
  3. In the Source section, change the branch to main (or master if that’s the default), and make sure the folder is set to / (root).
  4. Click Save.

5. Access Your Website

  • After enabling GitHub Pages, GitHub will provide you with a URL to access your site. It will be something like:
